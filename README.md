## 📋 Sobre o Projeto

O objetivo foi desenvolver um mini site funcional aplicando conhecimentos básicos de:

- **HTML** — estruturação semântica do conteúdo
- **CSS** — estilização, layout e efeitos visuais
- **JavaScript** — interatividade e validação

O tema escolhido foi um **portfólio pessoal**, apresentando habilidades, projetos acadêmicos e um formulário de contato.

---

## ✅ Requisitos Atendidos

### HTML
- [x] Estrutura semântica com `<header>`, `<main>` e `<footer>`
- [x] 1 imagem (foto de perfil)
- [x] 1 lista (`<ul>` de habilidades)
- [x] Links de navegação (menu com âncoras + GitHub externo)
- [x] 1 formulário simples (nome, e-mail e mensagem)

### CSS
- [x] Cores personalizadas com variáveis CSS (`:root`)
- [x] Alteração de fontes e espaçamentos
- [x] Layout organizado com Flexbox e CSS Grid
- [x] Efeitos visuais de hover nos cards, lista, botões e foto

### JavaScript
- [x] Alternância de tema claro/escuro
- [x] Alerta de boas-vindas
- [x] Validação completa do formulário com feedback visual
- [x] Scroll suave entre seções (bônus)

---

## 🚀 Funcionalidades

**🌙 Modo Escuro**
Alternância entre tema claro e escuro com transição suave, utilizando variáveis CSS e manipulação de classes via JavaScript.

**✨ Alerta de Boas-vindas**
Ao clicar em "Saiba mais", uma mensagem de boas-vindas é exibida ao usuário.

**📬 Formulário com Validação**
O formulário valida se todos os campos estão preenchidos e se o e-mail tem formato válido antes de exibir a mensagem de sucesso.

**🖱️ Scroll Suave**
Os links do menu navegam suavemente até as seções da página.

## 💡 Conceitos Aplicados

- Variáveis CSS com `:root`
- Flexbox e CSS Grid para layout responsivo
- `classList.toggle()` para alternância de tema
- `addEventListener` para scroll suave
- Regex para validação de e-mail
- Template literals com `${}` no JavaScript
- `backdrop-filter: blur()` para efeito glassmorphism no header
- Efeitos de hover com `transform` e `transition`
